Function ToBool

Parameters
$value - The object value.

Returns

The bool value.

Syntax

PHP
public static function ToBool($value): bool

Returns a value as bool.

Example

PHP
 // Returns a value as bool.
 private static function ToBool()
 {
   $text = "false";
   $value = LJC::ToBool($text);
   $result = $value ? "true" : "false";
 
   // result:
   // false
 }
