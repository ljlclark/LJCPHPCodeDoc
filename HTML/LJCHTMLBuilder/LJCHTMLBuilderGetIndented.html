Function GetIndented

Parameters
$text - The add text.

Returns

The potentially indented new text value.

Syntax

PHP
public function GetIndented(string $text): string

Gets a new potentially indented line.

Example

PHP
 $hb = new LJCHTMLBuilder();
 
 // Example Method:
 $result = hb->GetIndented("This text is NOT indented.");
 $hb->AddText($result);
 
 $hb->AddIndent(2);
 $hb->AddLine();
 $result = $hb->GetIndented("This text is indented.");
 $addIndent = false;
 $hb->Text($result, $addIndent);
 $result = $hb->ToString();
 
 // result:
 // This text is NOT indented.
 //     This text is indented.
