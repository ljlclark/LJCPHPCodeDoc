Function Log

Parameters
$lineNumber - The debug line number.
$valueName - The value name.
$value - The value.
$asObject - Indicates if outputting as an object.
$output - Indicates if value is output.

Syntax

PHP
public function Log(int $lineNumber, string $valueName, $value
, bool $asObject = false, bool $output = true): string

Outputs the value or object text.

Example

PHP
 // Outputs the value debug text.
 private static function OutputLog()
 {
   $methodName = "OutputLog";
 
   $valueName = "\$value";
   $value = "Test";
   $lineNumber = __line__;
   $output = LJC::OutputLog($lineNumber, $valueName, $value, output: false);
 
   // output:
   //
   // 123 $value = Test
 }
