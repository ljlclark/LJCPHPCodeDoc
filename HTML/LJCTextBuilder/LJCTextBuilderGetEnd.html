Function GetEnd

Parameters
$name - The element name.
$textState - The current text state values.
$addIndent - _ParamSummary_

Returns

The element end tag.

Syntax

PHP
public function GetEnd(string $name, LJCTextState $textState
, bool $addIndent = true): string

Gets the element end tag.

Remarks

GetEnd() is a "GetText" method which creates the element closing tag and returns the text value.

Example

PHP
 // Root Method Begin
 $textState = new LJCTextState();
 
 $tb = new LJCTextBuilder($textState);
 
 // Example Method:
 $result = $tb->GetEnd("p", $textState);
 
 // result:
 // </p>
